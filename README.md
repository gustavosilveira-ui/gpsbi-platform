# GPSBI Platform - Protótipo Web

Projeto React + Vite pronto para subir no Vercel.

## Como rodar localmente

```bash
npm install
npm run dev
```

## Como publicar no Vercel

1. Crie uma conta no Vercel.
2. Envie esta pasta para um repositório no GitHub.
3. No Vercel, clique em **Add New Project**.
4. Selecione o repositório.
5. Clique em **Deploy**.
